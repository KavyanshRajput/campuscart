CampusCart (AcroCart)
CampusCart is a peer-to-peer marketplace app built for students at Acropolis Institute. It allows us to buy, sell, and trade college essentials like old books, lab coats, calculators, and electronics directly with each other on campus.

Live Link: https://campuscart-9c677.web.app

Github Repo: https://github.com/KavyanshRajput/campuscart

Core Features
Acropolis Domain Lock: Only users with an @acropolis.in email ID can log in, keeping the marketplace secure and exclusive to our campus.

Real-Time Chat: A built-in messaging system to negotiate prices and plan meetups on campus without sharing phone numbers.

PWA Support: The app can be installed directly onto your phone's home screen or desktop as a standalone app.

Smart Search: Filter items easily by category, maximum budget, or how long the item has been used (usage age).

Tech Stack
Frontend: Next.js (App Router), React, CSS Modules

Backend & DB: Firebase Auth, Firestore (Real-time Database), Firebase Storage

Hosting: Firebase Hosting
